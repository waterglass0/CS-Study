# 5.1 복잡도
# 5.1.1 시간 복잡도
## 빅오 표기법
- 시간 복잡도
  - 문제를 해결하는 데 걸리는 시간과 입력의 함수 관계
  - 보통 빅오 표기법으로 나타냄
    - 입력 크기 n을 기준으로 하여 로직이 몇 번 반복되는지 나타낸 것
    - 필요 시간이 `10n^2 + n`이면, 시간 복잡도는 O(n^2)
    - 가장 영향이 큰 항의 상수 인자만 빼고 나머지는 무시한다.
  
## 시간 복잡도의 존재 이유
- 걸리는 시간이 짧은 효율적인 코드로 개선하기 위해

## 시간 복잡도의 속도 비교
- O(1) > O(logn) > O(n) > O(n^2)
- n이 커질수록 속도 차이가 커진다.
